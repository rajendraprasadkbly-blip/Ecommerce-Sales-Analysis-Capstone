# Acceptance Criteria

### Dashboard Functionality
- All charts must update when filters are applied.
- Date slicer must filter all visuals.
- KPIs must calculate correctly using DAX.

### Data Quality
- No leading or trailing spaces in text fields.
- All numeric fields correctly typed.
- Missing discounts replaced with 0.

### Visual Requirements
- Overview page must contain KPIs.
- Category insights must show clear comparison.
- Sales trend must show monthly view.
